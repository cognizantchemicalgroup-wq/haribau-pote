# Haribau Pote - Traditional Masala E-commerce Website

A beautiful, fully responsive e-commerce website for "Haribau Pote" - an authentic pote masala brand from Panvel, Maharashtra, India.

## Features

### Multi-Page Website
- **Home Page** - Hero section, features, popular products, testimonials
- **Products Page** - Complete product catalog with filtering options
- **About Page** - Brand story, timeline, values, team, and certifications
- **Recipes Page** - Traditional Maharashtrian recipes using Haribau Pote Masala
- **Contact Page** - Contact form, business information, FAQ, and map

### Key Features
- Fully responsive design (mobile, tablet, desktop)
- Shopping cart functionality with localStorage
- Product filtering system
- Interactive recipe modals
- Contact form with validation
- Newsletter subscription
- Smooth animations and transitions
- Back-to-top button
- Mobile-friendly navigation

### Design Highlights
- Beautiful Indian-themed color palette (Orange, Saffron gradients)
- Modern card-based layouts
- Engaging hero sections
- Professional product displays
- Interactive elements
- Optimized user experience

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with gradients, animations, flexbox, and grid
- **JavaScript (Vanilla)** - Interactive features, cart system, form handling

## How to Use

### Local Setup
1. Extract the files to a folder
2. Open `index.html` in any modern web browser
3. Navigate through the website using the menu

### Customization

#### Adding Product Images
- Replace placeholder backgrounds in CSS or add actual images
- Update the `.product-img-placeholder` sections in HTML

#### Modifying Products
- Edit product details in `products.html`
- Update prices, descriptions, and features
- Add new products by copying product card structure

#### Updating Contact Information
- Modify contact details in footer and contact page
- Update address, phone numbers, and email

#### Adding Real Recipes
- Add recipe images to the `images/` folder
- Update recipe data in `js/recipes.js`
- Expand the `recipesData` object with new recipes
